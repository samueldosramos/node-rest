# Node.js REST API

A simple REST API made with [Node.js](https://nodejs.org/).

## Install

Clone this project

```
git clone git@github.com:samuelramox/node-rest.git
```

Install the dependencies

```
npm i
```

Finally, run this project:

```
npm run start
```
